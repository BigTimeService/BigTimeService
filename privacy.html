# BigTime Tools 插件隐私政策 我们非常重视您的隐私。本插件仅用于本地读取和展示 OpenLoot 平台的用户资产数据，不会收集、存储或上传任何用户的个人信息、浏览历史、密码或 cookie 内容。 ## 收集信息 插件不会收集用户输入、浏览行为或账户凭证。插件仅通过 Chrome API 访问本地存储（storage）和 Cookie，用于判断用户登录状态。 ## 数据处理 所有处理逻辑在本地浏览器中完成，不会将数据发送到任何第三方服务器。我们不会追踪、分析或记录任何与用户身份相关的数据。 ## 权限使用说明 插件所需权限（如 `cookies`、`tabs`、`host_permissions`）仅用于实现功能，不作他用。详见 Chrome 插件权限声明。 ## 联系方式 如有疑问，请通过 GitHub 提交 issue 或邮件联系开发者。
